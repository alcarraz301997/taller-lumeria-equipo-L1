## Commands

## Architecture

- React 19 + Vite
- Bilingual content (en/es)
- Content loaded through loaders
- Language managed via LanguageContext

## Important Rules

- Use import type for type-only imports
- No enums
- No namespaces
- No parameter properties
- Remove unused code
