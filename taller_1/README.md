# 🖥️ Terminal Portfolio

Portafolio interactivo estilo terminal desarrollado con React + TypeScript + Material UI.

> Simula una experiencia CLI para explorar mis proyectos, habilidades y experiencia.

## 🚀 Getting Started

```bash
git clone https://github.com/tu-user/terminal-portfolio
cd terminal-portfolio
npm install
npm run dev
```

## ⚙️ Tech Stack

- ⚛️ React + TypeScript
- 🎨 Material UI (custom styling)
- ⚡ Vite
